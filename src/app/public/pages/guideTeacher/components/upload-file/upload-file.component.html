Subir un Objeto de aprendizaje

Cargar el archivo comprimido del Objeto de aprendizaje

Una vez registrados con el perfil de profesor podemos visualizar el menú, con las siguientes opciones: mi perfil, seguridad, mis objetos de aprendizaje, subir objeto de aprendizaje y salir.

A continuación, te explicaremos como puedes subir un objeto de aprendizaje.

Selecionar un archivo para cargarlo en el repositorio

Seleccionamos el archivo comprimido .zip referente al Objeto de Aprendizaje, se debe tener en cuenta que los objetos de aprendizaje deben estar bajo las especificaciones como IMS o SCROM para que se pueda subir el OA.

Se podrá observar un formulario para llenar los datos referentes a el objeto de aprendizaje.

Selecionar una foto de referencia en el formulario

Una funcionalidad adicional dentro del repositorio de objetos de aprendizaje es brindarte una lista de imágenes, las cuales son extraídas del mismo objeto de aprendizaje. Puedes elegir una imagen para que sea tu portada o imagen de referencia de este objeto de aprendizaje dentro del repositorio.

Dato caso se puede subir una imagen que se desee desde el propio ordenador.

Preguntas para la evaluación de accesibilidad

Dentro del formulario existe preguntas que nos ayudan a obtener una evaluación mas certera de los metadatos que tiene el objeto de aprendizaje.

Ya que de manera automática la plataforma detecta los recurso que posee el objeto de aprendizaje. Como se puede apreciar los recursos que se busca son: imágenes, párrafos, videos y audios.
